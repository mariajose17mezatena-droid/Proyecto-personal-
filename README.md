#Proyecto Personal 

STARS es una aplicación web desarrollada como proyecto personal, enfocada en ayudar a las personas a crear y gestionar rutinas de ejercicio de manera sencilla e inteligente. La plataforma 
utiliza inteligencia artificial para generar planes de entrenamiento personalizados según los objetivos y necesidades de cada usuario. 
Además, incorpora el uso de realidad aumentada para mostrar la ejecución correcta de los ejercicios, brindando una experiencia más interactiva y facilitando el aprendizaje de los movimientos.
La aplicación también permite organizar rutinas, registrar el progreso de cada entrenamiento y ofrecer recomendaciones para mantener la constancia y mejorar el rendimiento físico. 

El propósito del proyecto es brindar una alternativa accesible para quienes desean entrenar de forma segura y eficiente desde cualquier lugar. Este proyecto está dirigido principalmente a personas 
mayores de 15 años que buscan una forma económica y práctica de realizar actividad física sin depender de un gimnasio o de un entrenador personal. Asimismo, está pensado para personas con horarios 
ocupados que necesitan una herramienta flexible para ejercitarse en casa o en cualquier otro espacio disponible. 

Mediante STARS, los usuarios pueden acceder a rutinas personalizadas, mejorar su técnica y mantener hábitos saludables utilizando tecnologías innovadoras.
